# paradoxicalbot

A bot using seq2seq model using tensorflow.

--------

## Test The Bot
Use `bot.py` to test the bot.

## Usage

**Train**
```bash
python3 bot.py train
```

**Generate Text**
```bash
python3 bot.py generate <first word> <second word>
```

Example:
```bash
python3 bot.py generate i am
```
